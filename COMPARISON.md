## Getting Started

```bash
# Clone repository
git clone https://github.com/kRxZykRxZy/ServerControl

# Build
./build.sh

# Deploy on servers
./servercontrol

# Run control panel
./control/control
```

That's it! No accounts, no API keys, no cloud setup.
